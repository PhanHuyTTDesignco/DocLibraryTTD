Raise(String, Action<Document>, Boolean, TransactionStatus)

Hàm dùng để hỗ trợ lấy Document từ bên ngoài. Thực hiện mở đóng Transaction hoặc TransactionGroup.

Declaration
public static void Raise(string transactionName, Action<Document> actionDoc, bool isTransactionGroup = false, TransactionStatus transactionEnum = TransactionStatus.Commit)
Parameters
Type Name Description
System.String transactionName

Tên của transaction
System.Action<Document> actionDoc

Function có đầu vào là Document hoặc có thể viết truyền vào một Lambda expressions
System.Boolean isTransactionGroup

Kiểu của transaction. Mặc định sẽ = false sẽ là Transaction, nếu bằng true thì sẽ mở TransactionGroup
TransactionStatus transactionEnum

Trạng thái khi đóng transaction.
Examples

Khi mở Transaction và trạng thái đóng Transaction là Commit

ExternalEventUtils.Raise("transactionName" ,(Document doc) =>{
//code logic
});

Khi mở Transaction và trạng thái đóng Transaction là RollBack

ExternalEventUtils.Raise("transactionName" ,(Document doc) =>{
//code logic
}, transactionEnum : RollBack );

Khi mở TransactionGroup và trạng thái đóng TransactionGroup là Commit

ExternalEventUtils.Raise("transactionName" ,(Document doc) =>{
doc.TransactionStart("transaction Name 1" , doc =>{
// code logic khi mở transaction 1
});
doc.TransactionStart("transaction Name 2" , doc =>{
// code logic khi mở transaction 2
});
},isTransactionGroup : true );
